# 🛒 E-Commerce Frontend (Amazon Clone)

> 🌐 **Live Demo:** [ahmad123-uet.github.io/Ecommerce-Frontend-Amazon-Clone](https://ahmad123-uet.github.io/Ecommerce-Frontend-Amazon-Clone/)
> 🔗 **Repository:** [github.com/ahmad123-UET/Ecommerce-Frontend-Amazon-Clone](https://github.com/ahmad123-UET/Ecommerce-Frontend-Amazon-Clone)

A responsive front-end clone of the Amazon homepage, built with **HTML5** and **CSS3**. This project focuses on layout structuring, Flexbox, and core responsive web design fundamentals.

<!-- ![Project Screenshot](screenshot.png) -->

## 📝 Overview

This project recreates the core layout of the Amazon homepage — navigation bar, category panel, hero banner, and product category grid — using only HTML and CSS, without any frameworks or libraries.

## ✨ Features

- Amazon-style navigation bar (logo, delivery location, search bar, account, cart)
- Category navigation panel
- Hero/banner section
- Responsive product category grid (Electronics, Clothes, Furniture, Beauty, Pet Care, and more)
- Multi-column footer with informational links

## 🛠️ Tech Stack

| Technology | Purpose |
|---|---|
| HTML5 | Page structure and semantics |
| CSS3 (Flexbox) | Layout and responsive styling |
| [Font Awesome](https://fontawesome.com/) | Icons (search, cart, location, etc.) |

## 🚀 Getting Started

No installation or build tools required — just clone the repo and open `index.html` in your browser.

Or view it directly via the **[Live Demo](https://ahmad123-uet.github.io/Ecommerce-Frontend-Amazon-Clone/)** — no setup needed.

## 📚 Key Learnings

- Structuring a multi-section page with semantic HTML
- Building responsive layouts using Flexbox
- Styling reusable UI components (navbar, cards, grid)
- Working with external icon libraries
